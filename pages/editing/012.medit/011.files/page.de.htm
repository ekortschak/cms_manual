Bestehende Dateien

Bestehende Dateien werden in einer Liste angezeigt. Folgende Funktionen stehen zur Verfügung:

Icon Funktion
✗ Datei löschen
Datei ausblenden (Kontext-abhängig)
Datei einblenden

Dateien erstellen

Dateien werden grundsätzlich nur dann überschrieben, wenn die Option "Bestehende Dateien überschreiben" aktiviert ist.

Typ Funktion
Ini-Datei(en) Erstellt für den aktuellen Menüpunkt eine Standard-Ini-Datei. Sollte schon eine bestehen, werden deren Werte in die neue Datei übernommen.
System-Datei Aufgrund des Namens werden System-Dateien entsprechend ihrer Definition in die Seite eingebaut. Funktionsweise wird weiter unten beschreiben.
Andere Datei Es können beliebige Dateien erstellt werden. Wenn diese auch angezeigt werden sollen, müssen sie über ein PHP-Script in die Seite eingebunden werden.
