CMS = Content Management System

Freeware - ohne Einschränkungen

Mit diesem CMS möchte ich DANKE sagen an alle

  • die ihrerseits Freeware ins Netz gestellt haben
  • mit deren Tools ich arbeiten durfte (sogar MS)
Haftungsausschluß

Diese Software wird so zur Verfügung gestellt, wie sie ist. Sie wurde nach bestem Wissen und Gewissen auf Grund jahrelanger Erfahrung erstellt. Sie ist aber weder "sicher" noch "fehlerfrei".
Die Verwendung erfolgt auf eigenes Risiko.

Features

Allgemeine Funktionen

Dieses CMS hat mit anderen gemeinsam:

  • Verwaltung von Webseiten
  • Auf modularen Vorlagen basierende Ausgabe
  • Unterstützung von Datenbanken
Spezielle Funktionen

Dieses CMS unterscheidet sich von anderen v.a. auch dadurch:

  • Freeware für jedermann, auch für kommerzielle Zwecke jeder Art
  • (fast) vollständig Datei-basiert
  • Verwaltung von Datenbanken: derzeit MySQL, sqLite (im Test)
  • schlanke Ausgabe-Dateien
  • größtmögliche Modularität
  • beliebige Erweiterbarkeit
Was es nicht ist
  • kein WYSIWYG Editor (minimale HTML-Kenntnisse erforderlich)
Verwendete Programmiersprachen
  • HTML, CSS, JS
  • PHP
  • SQL

Unterstützung

Wer das Projekt unterstützen möchte, kann sich per EMail an den Autor wenden. Solltet ihr coole Zusatz-Features hinzufügen, würde es mich freuen, wenn ihr mir diese auch zur Verfügung stellen wolltet.
