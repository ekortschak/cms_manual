Alle Daten können vor dem Speichern bzw. Löschen überprüft werden.
Jede Änderung kann Folgebuchungen triggern.

Dazu wird im Ordner dbase/validate für jede Tabelle mit besonderen Validierungsregeln eine Ini-Datei angelegt, z.B. dbase/validate/tblxyz.php.

chkVal($inf, $fld, $val)
$inf = alle Feldeigenschaften von $fld
$fld = Feldname
$val = beabsichtigter Wert
b4Xxx($vls, $qid)
$vls = alle Felder des geänderten/neuen Datensatzes
$qid = ID des zu ändernden Datensatzes (wo sinnvoll)
onXxx($vls, $qid)
$vls = alle Felder des geänderten/neuen Datensatzes
$qid = ID des zu ändernden/neuen Datensatzes
