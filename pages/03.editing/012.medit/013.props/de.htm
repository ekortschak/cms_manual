Tab-Eigenschaften

Im Allgemeinen werdend die Tab-Eigenschaften im Tab-Editor festgelegt. Weil es aber einfacher ist, den Startpunkt direkt im Menü auszuwählen, ist diese Funktion bei den Seiten-Eigenschaften untergebracht.

Durch Setzen des Häkchens wird der ausgewählter Menüpunkt zum Startpunkt. Früher gesetzte Startpunkte werden überschrieben.

Seiten-Eigenschaften

Eigenschaft Funktion
UID Eindeutiger Bezeichner, der den Menüpunkt eindeutig identifiziert. Die UID wird benötigt, um die jeweilige Seite aufzurufen. Wenn die ID nicht eindeutig ist, wird jeweils der letzte Menüpunkt mit der gleichen ID angesprungen.
Ordner-Inhalt Legt fest, wie der Seiteninhalt anzuzeigen ist.
Nicht drucekn Wird verwendet, wenn die aktuelle Seite nicht im Sammeldokument aufscheinen soll.
Titel Titel im Menü
Überschrift Titel über dem Seiteninhalt
Schlüsselwörter Derzeit nicht in Verwendung
Seitenvorschub Wird verwendet, wenn im Sammeldokument vor der aktuellen Seite ein Seitenvorschub gewünscht sein sollte.

Ordner-Inhalte

Es gibt vordefinierte Darstellungen für Menüpunkte, so daß für diese jeder Programmieraufwand wegfällt und durch die Definition von geeigneten Parametern ersetzt wird.

Typ Beschreibung
default Das CMS entscheidet selbständig, ob es sich bei dem Menüpunkt um einen einfache Menüpunkt, eine Seite oder eine Kombination aus beiden handelt.
file upload Es wird ein Formular zum Hochladen von Dateien angeboten. Vorgaben zur max. Dateigröße und Dateitypen sind möglich.
file download Alle Dateien im Verzeichnis werden zum Download angeboten, sofern die Dateitypen mit den spezifizierten Erweiterungen übereinstimmen.
file viewer veraltet
mime type viewer Alle Dateien werden entsprechend des Dateityps in einem dafür geeigneten Add-on angezeigt. So werden PDF-Dateien in einem PDF-Viewer angezeigt, Bilder und Multimedia-Inhalte in den dafür geeigneten Erweiterungen des Browsers.
ip camera Es wird das Bild einer verbunden IP-Kamera angezeigt.
internal redirection Wenn Ihnalte an mehreren Stellen im Menü angeboten werden sollen, empfiehlt es sich, diese nicht mehrmals zu definieren, sondern einfach auf jenes Verzeichnis zu verweisen, wo die Daten tatsächlich liegen.
collection Alle Unterordner werden gesammelt und in eine Navigations-Tool integriert.
